# ASReview Screenshot Instructions

To display the ASReview interface preview in the Literature Review tab, please add a screenshot image:

## Filename
`asreview_screenshot.png`

## Location
Place the file in this directory: `/assets/`

## Recommended Specifications
- **Format**: PNG (for best quality) or JPG
- **Dimensions**: 1200px wide or larger (maintains readability)
- **Content**: Full ASReview interface showing:
  - Project list or review screen
  - Key interface elements (navigation, buttons, etc.)
  - Example of the review workflow

## How to Capture
1. Access your ASReview service at the configured URL
2. Navigate to a representative screen (e.g., project overview or review interface)
3. Take a full-window screenshot
4. Crop if needed to focus on the main interface
5. Save as `asreview_screenshot.png`
6. Place in this `/assets/` directory

## Fallback Behavior
If the screenshot is not provided, users will see a placeholder message with instructions to add the image.

## Example ASReview Views to Screenshot
- **Project Dashboard**: Shows available projects and options
- **Review Screen**: Active learning interface with paper selection
- **Analytics View**: Progress metrics and completion statistics

The screenshot helps administrators and users understand what ASReview looks like before accessing it directly.
